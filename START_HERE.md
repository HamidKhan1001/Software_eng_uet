## 🌍 Deploy to Production

You can deploy in 20 minutes using:

### **Frontend**: Vercel (GitHub + click deploy)
```
1. Connect GitHub repo
2. Add REACT_APP_API_URL env var
3. Deploy (automatic)
4. Live at https://yourdomain.vercel.app
```

### **Backend**: Railway or Render
```
1. Connect GitHub repo  
2. Add DATABASE_URL and other env vars
3. Deploy
4. Live at https://your-api.railway.app
```

## 🛠️ Technology Stack

- **Frontend**: React 19 with React Router
- **Backend**: Node.js + Express 5
- **Database**: PostgreSQL (Neon free cloud DB)
- **Auth**: JWT tokens (7-day expiry)
- **Email**: Gmail SMTP (Nodemailer)
- **Hosting**: Vercel (frontend) + Railway/Render (backend)
- **Styling**: CSS3 with Mobile-First design
- **Security**: Bcrypt passwords, CORS, Rate limiting
