# Members Photos Directory

Add member photos here to display them on the home page.

## File Naming Convention

Name files according to the member names in the format:
- `HamidNaeem.jpeg` - for Hamid Naeem
- `malak-saad-khan.jpeg` - for Malak Saad Khan
- `muhammad-adam.jpeg` - for Muhammad Adam
- `muhammad-umar.jpeg` - for Muhammad Umar
- `hammad-khan.jpeg` - for Hammad Khan
- `waleed-khan.jpeg` - for Waleed Khan
- `mustaqim-khan.jpeg` - for Mustaqim Khan
- `zuhaib.jpeg` - for Zuhaib
- `muhammad-ahmad-mushtaq.jpeg` - for Muhammad Ahmad Mushtaq
- `abdul-samad.jpeg` - for Abdul Samad
- `hamza-taif.jpeg` - for Hamza Taif

## Image Requirements

- Format: JPEG or PNG
- Size: Recommended 300x300px minimum
- Square images work best
- The system will show initials if no image is provided

## See Also

Check `src/pages/Home.js` for the `getImagePath` function to see the filename mappings.
